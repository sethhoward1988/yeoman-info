Managing Dependencies with Bower

Bower automates the task of finding, downloading, and updating web assets like JavaScript libraries, CSS stylesheets, or hosted images.

Searching Packages

Search the bower registry with the following commands: 

              # search for a dependency by name
              bower search _name_

              # list all packages in bower registry
              bower search

Installing and Updating Dependencies

The power of bower comes from automatically installing and updating assets. It saves you from needing to go manually download, unzip, copy into resource directory, and point your site to it. Bower can inject your new dependencies right into your pages and even register packages with tools like RequireJS. 

              # install one or more dependencies
              bower install _name_ _name2_

              # update a dependency
              bower update _name_

              # remove a dependency
              bower uninstall _name_
