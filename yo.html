Scaffolding Projects with Yo

The major contribution of the Yeoman project is the yo script which scaffolds out new projects and coordinates the configuration of our other tools Bower and Grunt. Yo accomplishes its work by using scaffolding formulas called "generators" which dictate the directories and config files to create. You can find special generators for general webapps, Ruby on Rails, Django, AngularJS, Backbone.js, Ember.js, Jekyll, and many more.

Using the Webapp Generator

Get started by installing and using the generator-webapp with the following code in your terminal: 

              # install webapp generator for yo
              npm install -g generator-webapp

              # create a folder for your project
              mkdir myproject
              cd myproject

              # use webapp generator to scaffold your project
              yo webapp

This will scaffold a new web application in your myproject/ directory. When you first run yo webapp you will be greeted by the Yeoman himself and asked to choose from a list of libraries you might want to include like SASS, Twitter Bootstrap, or Modernizr. Below is a screenshot of the Yeoman:

In the background, Yo will just setup and run Bower for the extra dependencies.

Finding Community Generators

To find generators built by the community, you can consult the official list by Yeoman at the folling site: http://yeoman.io/community-generators.html

You can also explore the npm packages and search for ones containing "generator" or "yeoman" at the npm registry: https://npmjs.org/
Alternatively, you can use the npm search command: 

                npm search yeoman
